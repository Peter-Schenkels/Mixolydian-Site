Boulder Hall Manager - Path Editor & Outline Renderer

Posted on 4-9-2023 at 02:22 am

Path Editor

Hello, it's 2 am again, so you know what that means! Time to post an update about the latest changes I made to my management game. I made a programming interface for seperate paths, which are visualised by the outline. In the next update I will probably add some handles for editing already placed grips.

Outline Shader

As you can see I also implemented my own Outline shader, which took some hours to make. I don't have alot of experience creating shaders so this was a fun way to learn more about them and the unity render pipeline. In short I created a custom post processing effect and passed a render texture which contains the meshes of objects that need to be rendered. After that I do some edge detection in the shader and sample 4 points around the center point of the shader pass. The line width determines how far those points are from the center. For now I'm very happy with the results and what I learned
