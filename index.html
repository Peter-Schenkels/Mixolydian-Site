Hello and welcome to my website!

My name is Peter Schenkels, and I am a game programmer currently working at Ronimo Games. I recently graduated with a degree in Computer Science from HU University of Applied Sciences. During my studies, I gained experience working on various projects using programming languages such as C++, C#, and Python.

One of my recent accomplishments was contributing to the development of software for a Low Powered Sensor that measures earthquakes at my university's professorship for Microsystems. However, my passion for game programming led me to my current position at Ronimo Games.

Outside of work, I continue to pursue my creative hobbies, including music production and working on personal software-related projects. Thank you for visiting my website, and I hope you find the information here helpful!

Game Development Highlight reel

From the end of 2019 to the beginning of 2022, I had the opportunity to work on a variety of game development projects. While some of these games were created as part of my school projects, others were personal passion projects.

As a game developer, I believe that games are one of the best mediums for creative expression. I am passionate about using my skills to bring immersive experiences to players, and I am constantly seeking new opportunities to work on exciting game development projects.

I hope that you enjoy this highlight reel and get a sense of my passion for game development. Thank you for watching!

Software

Traffic Jam

A quick, fun, arcadey and dark game, where you play as traffic control, but the goals is the opposite of what traffic control is supposed to do. This game was made during the GMTK 2023 Game Jam. It was alot of fun doing this with my co-workers. I did some programming and made most of the art.

Musical 3D Printer

This script is a fun and easy way to convert MIDI files into GCode files that can be executed by a 3D printer. The script currently supports playing only one note at a time, but it can still produce cool results.

"3D music" ~W. Visser

Jinkel Delivery

An arcade-style 2D side-scrolling game! In this game, you take on the role of a pizza delivery person, riding your scooter through a procedurally generated world filled with hilarious visuals. This game was created by me as part of my minor in game development in Finland, using C# and Unity. From the programming to the assets, I handled every aspect of its development.

Arkel Craft

This Minecraft-inspired game is a bare-bones clone created using Unity3D. The goal of the project was to learn about terrain and mesh generation through hands-on experience.

Karkel Lang

As part of a school assignment, I created an interpreted and compiled programming language using Python. The language is designed to be functional and can be interpreted or compiled to arm assembly. This project was an excellent opportunity to deepen my understanding of programming languages and their inner workings. More information about the language, including its syntax and usage, is available on my GitHub repository.

Geometry Dash Computer Vision

For a school project, I tackled a challenging problem using computer vision. The goal was to detect obstacles and platforms in the popular game Geometry Dash, using classical computer vision techniques. After conducting extensive research and testing.

Twits Desktop

Say goodbye to the discontinued Twitch desktop app and hello to this app! This app is designed to provide the same functionality as the original Twitch desktop app, including streaming your favorite streamers and live notifications. Created using C# and WPF.

Soap

A small messenger client/server for talking with friends over the internet. I learnt alot about c# and sockets while making this project. This messenger has basic features like sending messages, sending images, setting your own username, profile picture and color. More features like sending different files are coming up!

Hobo Simulator

I was part of a group project in Finland where we created an exciting game together. In the project, I took charge of the movement mechanics and created most of the assets, including the Hobo model/animation, the kitchen, and the music. Additionally, I developed a script for the Hobo movement and sound effects.

Lasertag

For a school project, my team and I were tasked with developing a laser tag game from scratch using C++ and Arduino Due. As part of the team, I contributed to the project by implementing the Infra Red receiver and sender components. These vital components were responsible for detecting and transmitting laser tag signals during gameplay. Through this project, I gained valuable experience in C++ programming and hardware development using Arduino Due.

Port-Forward

As part of a school project, my team and I developed an exciting 2D game using C++ and SFML. Our game was a unique take on the popular Portal franchise, where players could shift between dimensions to solve puzzles with their portal gun. In the project, I took charge of resource management, level editor development, basic physics implementation, and other inner game engine workings. Additionally, I created most of the sprites that brought our game to life.
