Software

Games and other software I made.

Lasertag

A school project were we had to develop a lasertag game from scratch using the infrared protocol. My contributions to this project are the implementations of the IR receiver & sender.

Port-Forward

Another school project were we had to develop a game in sfml using c++. We went for a 2d portal clone with a twist were you could shift between dimensions to solve the puzzles with your portal gun. I was in charge of all of the resource management, level editor, basic physics and other inner game engine workings. I also designed and created most of the sprites.

Soap

A small messenger client/server for talking with friends over the internet. This leans more towards the passion project side. I learnt alot about c# and sockets while making this project. This messenger has basic features like sending messages, sending images, setting your own username, profile picture and color. More features like sending different files are comming up!
