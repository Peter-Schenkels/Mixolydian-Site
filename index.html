About me:

Hello and welcome to my website! My name is Peter Schenkels and I'm a Computer Science student at the HU University of Applied Sciences. I have experience working with project using C++, C# and Python. My latest contribution can be found at my University's professorship for Microsystems. I helped develop software for their Low Powered Sensor for measuring Earthquakes! In my free-time I produce music and work on games and other software related projects.

Software

Games and other software I made.

Game Development Highlight reel

Here are some highlights of my Game Dev adventures from end of 2019 till beginning of 2022. Most of these games were projects for school. Some of them where passion projects. I hope that in future I'm able to work on more games since I think games are the best way the express your self creativly and meaningfuly.

Musical 3D Printer

A fun easy script that turns a midi file into a GCode file which a 3D printer can execute/play. currently only plays one note at the same time

"3D music" ~W. Visser

Jinkel Delivery

A fun arcadey 2D sidescroller game where you have to deliver pizzas on your scooter in a proceduraly generated world. This was a game I did for my minor game development in Finland. It was made using C# and Unity. All the assets and programming were done by me.

Karkel Lang

An interpreted and compiled programming language. This was a school assignment. We had to write our own programming language in a functional way using python. The language can be interpreted or compiled to arm assembly. More infor is available on my github.

Geometry Dash Computer Vision

This was also a school project. The assignment was to solve a problem using computer vision. I wrote a script that could detect obstacles and platforms in the game: Geometry Dash, using classical computer vision.

Twits Desktop

A desktop app replacing the discontinued twitch desktop app. It currently supports streaming your favorite streamers and live notifications. Made in C# and WPF.

Soap

A small messenger client/server for talking with friends over the internet. I learnt alot about c# and sockets while making this project. This messenger has basic features like sending messages, sending images, setting your own username, profile picture and color. More features like sending different files are coming up!

Lasertag

A school project were we had to develop a lasertag game from scratch using C++ and arduino Dues. My contributions to this project were the implementations of the Infra Red receiver & sender.

Port-Forward

Another school project were we had to develop a game using c++ and sfml. We went for a 2d portal clone with a twist where you could shift between dimensions to solve the puzzles with your portal gun. I was in charge of resource management, the level editor, basic physics and other inner game engine workings. I also created most of the sprites.
